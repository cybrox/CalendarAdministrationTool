Administration

Die Administrationsfnuktion ermöglicht es einem Administrator das Profil jedes Benutzers einzusehen. Durch die Auswahl eines Benutzers und einem Klick auf das -Icon werden Kalenderdaten, Semester und Auswertungen aus Sicht des gewählten Benutzers gezeigt. Durch einen Klick auf den Menupunkt "Administrator" im Benutzermenu kann dieser Modus verlassen werden.

In der Benutzerliste können sämtliche registrierte Benutzer verwaltet so wie neue Benutzer hinzugefügt werden. es ist möglich den Namen, die E-Mail Adresse, das Passwort und die Gruppe eines Benutzers zu ändern. Bitte beachten Sie, dass ein Ändern der eigenen Gruppenzugehörigkeit Sie versehentlich vom System ausschliessen kann. Dies kann nur durch einen anderen Administrator oder direkten Datenbankzugriff wieder rückgängig gemacht werden.

In der Kategorieliste können sämtliche im Kalender eingetragenen Kategorien bearbeitet so wie neue hinzugefügt werden. Die Benutzer müssen beim Erstellen eines neuen Termins eine der hier eingetragenen Kategorien auswählen. Beim Auswerten der Noten eines Benutzers wird die Ansicht in die hier gelisteten Kategorien aufgelistet.
