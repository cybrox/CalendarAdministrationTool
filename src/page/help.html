Allgemeine Hinweise
Diese Software ist im allgemeinen so konzipiert, dass sie ohne weitere Instruktionen verwendet werden kann. Zusätzlich findet sich im unteren Teil dieser Seite eine ausführliche Beschreibung der Funktionen der Software welche zusätzlich zur Hilfe gezogen werden kann.
Sollte trotzdem ein Problem auftreten oder die Software einen Fehler ausgeben, wenden Sie sich bitte an ihren Administrator.

Benutzermenu
Durch einen Klick auf Ihren Benutzernamen in der linken oberern Ecke können Sie ihr Benutzermenu öffnen, darin finden Sie mehrere Menupunkte:

Seitennavigation
Unter ihrem Benutzermenu finden sie vier weitere Menupunkte. Durch einen Klick auf einen dieser Punkte gelangen sie in den jeweiligen Bereich des Kalendertools. Informationen zum Kalender und der Terminliste finden sie im nächsten Abschnitt, Informationen zur Semesterverwaltung im Abschnitt darunter.

Benutzerkonto sperren
Sie können als normaler Benutzer Ihr Benutzerkonto nicht sperren. Sollten sie ihr Konto blockieren oder löschen wollen, wenden Sie sich bitte an einen Administrator. Bedenken Sie, dass dieser Vorgang nicht einfach wieder rückgängig gemacht werden kann.

Benutzerkonto
Durch einen Klick auf diesen Menupunkt gelangen Sie in die Kontoverwaltung wo Sie alle Details Ihres Benutzerkontos einsehen können. Dort finden Sie Ihre Benutzer-ID, Ihren Benutzertyp und Ihren Benutzernamen. Diese Werte hat der Administrator bei der Erstellung ihres Kontos festgelegt, sie können nicht verändert werden. Unterhalb finden sie Ihre eingetragene E-Mail Adresse so wie zwei Passwort Eingabefelder. Sie können ihre E-Mail Adresse ändern indem sie einfach in das E-Mail Feld klicken und eine neue Adresse eingeben. Sollten Sie auch das Passwort ändern wollen, geben sie ihr gewünschtes neues Passwort in beide Passwortfelder ein und speichern Sie die Einstellungen anschliessend mit einem Klick auf den "Änderungen speichern" Button unterhalb dier Passwortfelder.

Abmelden
Durch einen Klick auf diesen Menupunkt werden sie vom System abgemeldet. Wenn Sie vergessen sich an einem Computer abzumelden, melden sie sich einfach von einem anderen Computer aus an, der gespeicherte Sicherheitsschlüssel wird dadurch ungültig. Wenn Sie ihr Browserfenster schliessen wird ihr Sicherheitsschlüssel ebenfalls ungültig und Sie müssen sich beim nächsten Start erneut mit ihren Zugangsdaten anmelden.

Hilfe & Info
Der Menupunkt Hilfe und Info ruft diese Seite auf, auf der Sie eine allgemeine Hilfestellung zur Bedienung des Programmes so wie einige Informationen darüber finden.

Kalender
Unter dem Menupunkt "Kalender" finden sie ihren kompletten Terminkalender und eine Liste ihrer anstehenden Termine. Die Kalenderansicht stellt den Zeitraum eines Monates in einem Kalendergitter dar. Mit einem Klick auf das Feld eines Kalendertages werden weitere Details zu allen Terminen an diesem Tag so wie die Möglichkeit diese zu bearbeiten oder löschen oder neue Termine zu erstellen angezeigt.

Terminliste
Die Terminliste ist eine erweiterte Form der "Anstehende Termine" - Anzeige auf der Kalenderseite. Während auf der Kalenderseite nur Heute anstehende Termine angezeigt werden, zeigt die Terminliste ausnahmslos alle anstehenden Termine an. Als anstehenden Termin versteht sich hierbei jeder Termin dessen eingetragenes Datum Heute oder in der Zukunft liegt.

Termin anlegen
Im Kalender wird durch einen Klick auf einen Kalendertag ein Detailfenster des gewählten Tages angezeigt. In dieser Anzeige findet sich auch die Funktion "Termin Hinzufügen" mit der ein neuer Termin erstellt werden kann. Bei der Erstellung eines Termines müssen ein Name, eine Kurze Beschreibung so wie ein Datum eingegeben werden und der Termin einem Typ (Aufgabe oder Prüfung) und einer Kategorie zugeordnet werden.

Termin bearbeiten
In der Detailsansicht eines Kalendertages finden sich alle an dem betreffenden Tag anstehenden Termine. Durch einen Klick auf das Bearbeiten-Icon kann der Termin nachträglich bearbeitet werden. Die Bearbeiten Funktion steht auch in der Liste der Anstehenden Termine am rechten Rand zur Verfügung.

Termin löschen
Direkt neben dem Bearbeiten-Icon eines Termines in der Kalender-Detailansicht und der Terminliste findet sich auch die Funktion einen Termin zu löschen. Die Löschung eines Termines muss mit einer Sicherheitsabfrage bestätigt werden.

Semester
In der Semesteransicht finden Sie alle eingetragenen Semester. Sie können Semester selbst eintragen, bearbeiten und löschen um diese danach auszuwerten. Bei der Auswertung wird der Durchschnitt aller Noten innerhalb des gewählten Semesters angezeigt.

Semester anlegen
Durch einen Klick auf den Button "Semester hinzufügen" unter der Semesterliste kann ein neues Semester hinzugefügt werden. In dem Popup muss der Name des Semesters so wie das Start- und Enddatum eingetragen werden. Der Name kann hierbei beliebig gewählt werden, die Daten können entweder per Hand eingegeben werden oder mit einem Klick auf den Pfeil am hinteren Ende des Feldes aus einem kleinen Kalender ausgewählt werden.

Semester bearbeiten
Rechts neben jedem Semester findet sich ein Button mit dem "" - Icon. Durch einen Klick auf diesen Button öffnet sich das selber Popup wie bei der Erstellung eines Semesters mit eingetragenen Werten. Sie können diese Werte nun nach Belieben verändern und ihre Änderungen mit einem Klick auf den "Änderungen speichern" Button am unteren Ende des Popups speichern.

Semester löschen
Direkt neben dem Bearbeiten - Button jedes Termines findet sich ein "Löschen" - Button. Durch einen Klick auf diesen Button wird ein Popup angezeigt in dem sie die Löschung des Termins bestätigen müssen um versehentliches Löschen zu verhindern.

Auswertung
Die Terminliste ist eine erweiterte Form der "Anstehende Termine" - Anzeige auf der Kalenderseite. Während auf der Kalenderseite nur Heute anstehende Termine angezeigt werden, zeigt die Terminliste ausnahmslos alle anstehenden Termine an. Als anstehenden Termin versteht sich hierbei jeder Termin dessen eingetragenes Datum Heute oder in der Zukunft liegt.

Semester auswerten
Durch einen Klick auf den Auswerten - Button wird eine Übersicht aller Noten im betreffenden Semester angezeigt. Der Notenverlauf wird graphisch dargestellt, während der Durchschnitt und sämtliche Einzelnoten in einer kleinen Liste unterhalb des Graphen angezeigt werden.
